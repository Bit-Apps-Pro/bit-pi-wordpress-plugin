== API Usage ==

Bit Pi is a powerful integration tool that uses platforms API to send and receive data with users permission.
some operations are done by in frontend(Javascript) and some are done by in backend(PHP). Bit Pi did not collect any of integration data from users.
But collect some error, logs, and usage data to improve the plugin.
currently, we are using the following platforms API.

[Google Sheets API](https://www.google.com/sheets/about/)
[Google Drive API](https://www.google.com/drive/)
[Google Calendar API](https://workspace.google.com/products/calendar/)
[Elementor Form API/WP Hooks](https://elementor.com/features/form-builder/)
[Bit Form API/WP Hooks](https://bitapps.pro/bit-form/)
[Fluent Form API/WP Hooks](https://fluentforms.com/)
[Jot Form API](https://fluentforms.com/)
[Zoho CRM API](https://www.zoho.com/crm/)

== Privacy Policy ==
Bit Pi did not collect any of integration data from users without user consent.
In plugin support page we have info about this products and other products we developed and also have links of support and privacy policy of this products, which are 
load from [our server](https://wp-api.bitapps.pro/).

== Installation ==

1. Download the plugin.
2. From the WordPress Admin Panel, click on Plugins => Add New.
3. Click on Upload, so you can directly upload your plugin zip file.
4. Use the browse button to select the plugin zip file that was downloaded, and then click on Install Now.
5. Once installed, click “Activate”.
